In acest document sunt prezentati pasii in vederea rularii aplicatiilor create de mine cu scop personal.

Aplicatia "Overtime.py" este un mini-calculator de salariu. Acesta functioneaza dupa standardul American (adica daca orele lucrate pe saptamana depasesc 40h, surplusul de ore lucrate este platit 150%).

Aplicatia "zileIntarziere" are ca scop calcularea penalizarilor aferente zilelor de intarziere a achitarii taxei de scolarizare (pentru FSEGA, specializarea IE). Cu alte cuvinte, pentru fiecare zi de intarziere se adauga la taxa de scolarizare 0.04% penalizari pe zi.

Pasi pentru rularea aplicatiilor sunt similari:

1.deschiderea terminalului (cmd) si navigarea cu comanta "cd" pana la folderul aplicatiilor, adica "PythonApps".

2.in cmd se tasteaza comanta "python X.py" - X reprezinta numele aplicatiei care se doreste a fi rulata (in cazul de fata, Overtime sau zileIntarziere).
